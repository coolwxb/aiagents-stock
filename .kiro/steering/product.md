# AI股票分析系统 (AI Stock Analysis System)

## Overview
An AI-powered stock analysis platform for the Chinese A-share market. The system provides intelligent stock monitoring, analysis, and trading capabilities using DeepSeek AI models.

## Core Features
- **智能盯盘 (Smart Monitor)**: Real-time AI-powered stock monitoring with alerts
- **智策板块 (Sector Strategy)**: Sector-based analysis and strategy recommendations
- **智瞰龙虎 (Dragon Tiger Analysis)**: Analysis of institutional trading patterns (龙虎榜)
- **主力选股 (Main Force Selection)**: Identify stocks with significant institutional activity
- **GS策略 (GS Strategy)**: Custom trading strategy implementation and scheduling
- **持仓分析 (Portfolio Analysis)**: Portfolio management and analysis
- **量化交易 (Quantitative Trading)**: Integration with MiniQMT for automated trading

## Target Users
Chinese stock market investors and traders who want AI-assisted analysis and monitoring.

## Key Integrations
- DeepSeek AI for intelligent analysis
- Tushare/AKShare for market data
- MiniQMT (xtquant) for quantitative trading
- Email/Webhook (DingTalk/Feishu) for notifications
